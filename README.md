## Overview

This project implements a CNN-based face mask detection system that classifies facial images into three categories:

| Class | Description |
|---|---|
| `with_mask` | Person wearing a mask correctly |
| `without_mask` | Person not wearing a mask |
| `mask_weared_incorrect` | Person wearing a mask incorrectly |

Three models are trained and compared:
1. **Custom CNN** — built from scratch with 4 convolutional blocks
2. **MobileNetV2** — lightweight transfer learning model (ImageNet pre-trained)
3. **ResNet50V2** — deeper transfer learning model (ImageNet pre-trained)

---

## Dataset

- **Source:** [Face Mask Detection — Kaggle](https://www.kaggle.com/datasets/andrewmvd/face-mask-detection)
- **Total Images:** ~12,000
- **Classes:** 3
- **Image Type:** RGB JPG/PNG
- **Split:** 70% train / 15% validation / 15% test

## How to Run

### Option A — Kaggle Notebook (Recommended)

1. Go to [Kaggle](https://www.kaggle.com) and create a new Notebook.
2. Add the dataset: **Face Mask Detection** by andrewmvd.
3. Upload `face_mask_detection.ipynb` and run all cells.
4. Enable GPU accelerator: *Settings → Accelerator → GPU T4 x2*.

### Option B — Local / Colab

```bash
# 1. Clone this repository
git https://github.com/Shivaram-Balla/Face-Mask-Detection-Using-Convolutional-Neural-Networks
cd face-mask-detection-cnn

# 2. Install dependencies
pip install tensorflow matplotlib seaborn scikit-learn opencv-python

# 3. Download the dataset from Kaggle
#    Place it so DATA_DIR in the notebook points to:
#    ./data/images/{with_mask, without_mask, mask_weared_incorrect}/

# 4. Update DATA_DIR in cell 1 of the notebook, then run all cells
jupyter notebook code/face_mask_detection.ipynb
```

## Key Features

- Data augmentation (rotation, flip, zoom, brightness)
- Class-weighted loss to handle imbalanced data
- Two-phase transfer learning (head training → fine-tuning)
- Grad-CAM visualisations for interpretability
- ROC curves with AUC scores
- Error analysis on hardest misclassifications
- Automated figure saving
