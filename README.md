## Overview

This project implements a CNN-based face mask detection system that classifies facial images into three categories:

| Class | Description |
|---|---|
| `with_mask` | Person wearing a mask correctly |
| `without_mask` | Person not wearing a mask |
| `mask_weared_incorrect` | Person wearing a mask incorrectly |

Three models are trained and compared:
1. **Custom CNN** вЂ” built from scratch with 4 convolutional blocks
2. **MobileNetV2** вЂ” lightweight transfer learning model (ImageNet pre-trained)
3. **ResNet50V2** вЂ” deeper transfer learning model (ImageNet pre-trained)

---

## Dataset

- **Source:** [Face Mask Detection вЂ” Kaggle](https://www.kaggle.com/datasets/andrewmvd/face-mask-detection)
- **Total Images:** ~12,000
- **Classes:** 3
- **Image Type:** RGB JPG/PNG
- **Split:** 70% train / 15% validation / 15% test

## How to Run

### Option A вЂ” Kaggle Notebook (Recommended)

1. Go to [Kaggle](https://www.kaggle.com) and create a new Notebook.
2. Add the dataset: **Face Mask Detection** by andrewmvd.
3. Upload `face_mask_detection.ipynb` and run all cells.
4. Enable GPU accelerator: *Settings в†’ Accelerator в†’ GPU T4 x2*.

### Option B вЂ” Local / Colab

```bash

git clone https://github.com/<your-username>/face-mask-detection-cnn.git
cd face-mask-detection-cnn
```
